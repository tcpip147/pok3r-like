A program that map a input from keyboard to another.

From | To
:---: | :---:
`CapsLock + Q` | `Insert`
`CapsLock + A` | `Delete`
`CapsLock + W` | `Home`
`CapsLock + S` | `End`
`CapsLock + E` | `PgUp`
`CapsLock + D` | `PgDn`
`CapsLock + U` | `Backspace`
`CapsLock + O` | `Enter`
`CapsLock + I` | `Up`
`CapsLock + J` | `Left`
`CapsLock + K` | `Down`
`CapsLock + L` | `Right`
`Ctrl + CapsLock` | original `CapsLock`
